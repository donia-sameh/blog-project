# Blog Admin — Project Context

## What is this?
A blog post management dashboard for admins. No authentication. Internal use only.

## Stack
- Frontend: React 18 + Vite · React Router · CSS Modules
- Backend: Node.js + Express
- Database: SQLite
- Dev: `npm run dev` at root (runs both servers)

## Current Features
- List all posts (with filters)
- Create a new post

## Missing Features
- Edit post
- Delete post

## Current Goal
Add edit and delete post functionality — incrementally, without redesigning anything.
